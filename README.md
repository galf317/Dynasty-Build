# Build-A-Superstar Dynasty

A mobile-friendly, single-player football career game.

## Features

- Choose QB, RB, or WR
- Build eight traits using player-style archetypes
- Generate a fictional team
- Simulate 17-game seasons
- Reach the playoffs and Super Bowl
- Earn MVP, Offensive Player of the Year, and Rookie of the Year
- Track career stats, championships, and progression
- Save progress locally on the player's device

## Publish with GitHub Pages

### 1. Create a repository

Create a new public GitHub repository, for example:

`build-a-superstar`

### 2. Upload these files

Upload everything from this folder, including:

- `index.html`
- `.nojekyll`
- `.github/workflows/deploy-pages.yml`
- `README.md`

Keep the folder structure exactly the same.

### 3. Enable GitHub Pages

Open:

**Repository → Settings → Pages**

Under **Build and deployment**, choose:

**Source: GitHub Actions**

### 4. Publish

Commit the files to the `main` branch.

GitHub Actions will deploy the game automatically.

Your link will be:

`https://YOUR-USERNAME.github.io/build-a-superstar/`

Replace `YOUR-USERNAME` with your GitHub username and use your actual repository name at the end.

## No login required for players

Only the repository owner needs a GitHub account to publish the game. Anyone with the final GitHub Pages link can play without signing in.
